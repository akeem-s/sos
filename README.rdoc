SOS is a simple, in progress app that allows users to call themselves so they can escape undesirable social situations

User manual
1. Navigate to sos site
2. type your phone number and submit
3. When your phone rings, say the following: 

  "Hello"..."Yes, this is he/she"..."Yes, yes, I have a moment to talk."

4. Whisper "sorry" to whoever you didnt feel like talking to
5. Walk away

Technologies

Ruby, Ruby on Rails, Twillio API
